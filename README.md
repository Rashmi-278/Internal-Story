# Web3 Nextjs ChakraUI Truffle Template

This is a boilerplate for Truflle with Nextjs having basic wallet connectivity feature.

Stack:
Web3modal,
Ethers.js,
Truffle,
ChakraUI,
Nextjs,
Fleek

Deployed on Fleek
